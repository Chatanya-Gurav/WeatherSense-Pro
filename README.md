# WeatherSense Pro

Advanced Data Science Mini Project

## Project Description
Weather forecasting web application built using Python Flask and WeatherAPI.

## Features
- Live weather data
- Hourly forecast
- 10-day forecast
- Sunrise / Sunset
- Premium UI design

## Technologies Used
- Python
- Flask
- HTML/CSS
- WeatherAPI

## Run Project

pip install -r requirements.txt
python premium_weather_flask_app.py
